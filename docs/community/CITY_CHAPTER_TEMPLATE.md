# City Chapter Template

Use this template to propose a public-safe city or regional chapter for the Rave Culture Field Guide.

Copy this file into a new document such as:

```text
docs/community/cities/DETROIT.md
docs/community/cities/CHICAGO.md
docs/community/cities/LONDON.md
```

Keep it respectful, sourced, and safe. This is a cultural education guide, not a directory of private parties.
