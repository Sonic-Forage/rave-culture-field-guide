# GitHub Community Prompt Pack

Use these prompts in issues, discussions, PR reviews, or agent tasks.

## City chapter builder

```text
Create a public-safe city chapter for the Rave Culture Field Guide.
City/region: <CITY>
Use only public sources and clearly label oral/local memory.
Include: one-paragraph overview, roots/timeline, sounds/styles, community values, safer-event context, public-safe landmarks/archives, public DJs/labels/radio/zines, open questions, sources.
Do not include private addresses, active underground locations, private contacts, unverified allegations, or anything that could put crews/attendees at risk.
Use tags: city-chapter, sources, public-safe, elder-review-wanted.
```

## DJ glossary improver

```text
Improve this DJ/rave term for beginners: <TERM>.
Explain: what it means, why it matters on a dancefloor, a beginner practice drill, common mistakes, related terms, and one source if possible.
Use tags: dj-guide, glossary, practice-drill.
```

## Source-backed correction

```text
Review this claim: <CLAIM>.
Find public sources, identify whether the current wording is accurate, propose a correction, and mark sensitivity as low/medium/high.
Do not overstate certainty. Preserve cultural attribution.
Use tags: correction, sources, public-safe.
```

## Launch proof reviewer

```text
Review this launch claim: <CLAIM>.
Map it to a repo path, URL, commit, verifier command, or say unsupported.
Reject claims about revenue, public posting, video upload, outreach, GPU work, training, or private media unless proof is present.
Use tags: launch-proof, closed-gate, verified.
```
